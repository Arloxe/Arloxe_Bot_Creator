About the forge

A browser-based Character Card V2 and lorebook editor.

Arloxe Card Forge is a local-first tool for creating, editing, importing, and exporting AI character cards and lorebooks.

What it does

This tool helps creators build Character Card V2 files for use with platforms and tools such as SillyTavern, Chub.AI, Botbooru, and compatible card systems.

The goal is to make character creation more comfortable without forcing creators to hand-edit JSON files unless they want to.

Local-first by design

The editor is designed to run directly in your browser. Card data, avatars, and drafts are handled locally unless the project is changed in the future to add optional online features.

Compatibility

The forge aims to support Character Card V2 JSON and PNG cards with embedded metadata. Some platforms may use extra fields or extensions, so compatibility will improve over time.
